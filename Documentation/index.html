A) Introduction - top

Ravitha HTML5 website template is a custom design package that is a visually stunning and professionally designed tool for Cleaning Services to showcase their work. The template is built on a modern and minimalist aesthetic, with clean lines and spacious layouts that put the focus on the images of the designer's work. The template includes multiple sections to showcase different areas of the expertise, including services, gallery, teams, and more. With its sleek and modern design, high-quality images, and easy-to-use contact form, the template is sure to impress and inspire clients looking to create beautiful, functional, and personalized living spaces. It has a modern, clean, creative & unique design based on the latest technology.

It contains Home Page layouts and About Page Layouts and 04+ valid HTML5 page templates designs and different Blog styles. B) How to Edit HTML Files? - top

We have included 08+ custom HTML templates like Home Page styles, about styles and gallery page style and different Blog pages Styles etc. Please open any HTML files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want.

C) How to Edit CSS Files? - top

We have included some custom CSS styles like style.css (default). Please open any CSS files with a text editor like Dreamweaver, Notepad or Notepad++ and edit any lines what you want. For example if you want to edit your banner images open style.css and look at ".banner-sec::after" for banner image and change your styles.

 

D) How to use Color Schemes? - top

Please opencustom.css file from letsclean/assets/css folder with a text editor and build your own colors. #01d0cc this is our primary color, you can search and replace all to your new color code.

 

E) How to use On Load Animation? - top

We are aos.js on load animation for our website. you can edit them by simply adding or changeing the predefined classes name.
